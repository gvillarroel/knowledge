    <h1 id="introduction-to-ai-in-bigquery">Introduction to AI in Big<wbr>Query<devsite-actions><devsite-feature-tooltip class="devsite-page-bookmark-tooltip nocontent inline-block" id="devsite-collections-dropdown">

</h1>
    
<aside class="note"><strong>Note:</strong><span> This feature may not be available when using reservations that are created with
  certain BigQuery editions. For more information about which
  features are enabled in each edition, see <a href="/bigquery/docs/editions-intro">Introduction to
  BigQuery editions</a>.</span></aside>
<p>BigQuery offers various AI capabilities that let you do the following:</p>

<ul>
<li>Do predictive machine learning (ML).</li>
<li>Run inference against large language models (LLMs) such as Gemini.</li>
<li>Build applications using embeddings and vector search.</li>
<li>Use built-in agents to assist with coding.</li>
<li>Create data pipelines.</li>
<li>Access BigQuery functionality with agent tools.</li>
</ul>

<h2 id="machine_learning"><span class="devsite-heading">Machine learning</span></h2>

<p>With BigQuery ML, you can train, evaluate, and run inference on
models for tasks such as time series forecasting, anomaly detection,
classification, regression, clustering, dimensionality reduction, and
recommendations.</p>

<p>You can work with BigQuery ML capabilities through the
Google Cloud console, the bq command-line tool, the REST API, or in
<a href="/bigquery/docs/notebooks-introduction">Colab Enterprise notebooks</a>.
Because BigQuery ML lets SQL practitioners use
existing SQL tools and skills to build and evaluate models, it democratizes ML
and speeds up model development by bringing ML to the data instead of requiring
data movement. You can use BigQuery ML to help you with the
following types of ML tasks:</p>

<ul>
<li>
<a href="/bigquery/docs/model-overview">Create and run ML models</a>
by using GoogleSQL queries.</li>
<li>
<a href="/bigquery/docs/create-notebooks">Create Colab Enterprise notebooks</a>
to perform ML workflows. Notebooks
let you use SQL and Python interchangeably, and use any AI or ML Python
libraries for your development.</li>
<li>Understand the results of your predictive ML models with
<a href="/bigquery/docs/xai-overview">explainable AI</a>.</li>
<li>Use the <code>TimesFM</code>, <code>ARIMA_PLUS</code>, and <code>ARIMA_PLUS_XREG</code> models to perform
<a href="/bigquery/docs/forecasting-overview">forecasting</a> and
<a href="/bigquery/docs/anomaly-detection-overview">anomaly detection</a>
on time series data.</li>
<li>Generate insights about changes to key metrics in your multi-dimensional data
with <a href="/bigquery/docs/contribution-analysis">contribution analysis</a>.</li>
</ul>

<p>To learn more, see the
<a href="/bigquery/docs/bqml-introduction">Introduction to ML in BigQuery</a>.</p>

<h2 id="ai_functions"><span class="devsite-heading">AI functions</span></h2>

<p>BigQuery offers various SQL functions that you can use for
AI tasks such as text generation, text or unstructured data analysis, and
translation. These functions access Gemini and
partner LLM models available from Vertex AI, Cloud AI APIs, or
built-in BigQuery models to perform these tasks.</p>

<p>There are several categories of AI functions:</p>

<ul>
<li>
<p><strong>Generative AI functions.</strong> These functions help you perform tasks such as
content generation, analysis, summarization, structured data extraction,
classification, embedding generation, and data enrichment. There are two
types of generative AI functions:</p>

<ul>
<li>
<a href="/bigquery/docs/generative-ai-overview#general_purpose_ai">General-purpose AI functions</a>
give you full control and transparency on the choice of model, prompt,
and parameters to use.</li>
<li>
<a href="/bigquery/docs/generative-ai-overview#managed_ai_functions">Managed AI functions</a>
offer a streamlined syntax for routine tasks such as filtering, rating,
and classification. BigQuery can choose a model for you
to optimize for cost and quality.</li>
</ul>
</li>
<li>
<p><strong>Task-specific functions.</strong> These functions help you use Cloud AI APIs for
tasks such as the following:</p>

<ul>
<li><a href="/bigquery/docs/ai-application-overview#natural_language_processing">Natural language processing</a></li>
<li><a href="/bigquery/docs/ai-application-overview#machine_translation">Machine translation</a></li>
<li><a href="/bigquery/docs/ai-application-overview#document_processing">Document processing</a></li>
<li><a href="/bigquery/docs/ai-application-overview#audio_transcription">Audio transcription</a></li>
<li><a href="/bigquery/docs/ai-application-overview#computer_vision">Computer vision</a></li>
</ul>
</li>
</ul>

<p>For more information, see
<a href="/bigquery/docs/ai-application-overview">Task-specific solutions overview</a>.</p>

<h2 id="search">Search</h2>

<p>BigQuery offers a variety of search functions and features to
help you efficiently find specific data or discover similarities between data
including multimodal data.</p>

<ul>
<li><p><strong>Text search.</strong> You can use the
<a href="/bigquery/docs/reference/standard-sql/search_functions#search"><code>SEARCH</code> function</a>
to perform tokenized search on unstructured text or semi-structured <code>JSON</code>
data. You can improve search performance by creating a
<a href="/bigquery/docs/search-index">search index</a>, which lets
BigQuery optimize queries that use the <code>SEARCH</code> function, as
well as other functions and operators.
For more information, see <a href="/bigquery/docs/search">Search indexed data</a>.</p></li>
<li>
<p><strong>Embedding generation.</strong> Embeddings are high-dimensional numerical vectors
that represent entities such as text or images, and are often generated by ML
models. You can
<a href="/bigquery/docs/vector-search-intro">generate multimodal embeddings</a> by using
models provided by or hosted on Vertex AI, or by using models
imported and run in BigQuery.</p>

<p>You can also have BigQuery automatically maintain a column of
embeddings by enabling
<a href="/bigquery/docs/autonomous-embedding-generation">autonomous embedding generation</a>
(<a href="https://cloud.google.com/products#product-launch-stages">Preview</a>).</p>
</li>
<li>
<p><strong>Vector search.</strong> You can use the
<a href="/bigquery/docs/reference/standard-sql/search_functions#vector_search"><code>VECTOR_SEARCH</code> function</a>
to search embeddings to find semantically similar items. You can use the
<a href="/bigquery/docs/reference/standard-sql/bigqueryml-syntax-ai-search"><code>AI.SEARCH</code> function</a>
(<a href="https://cloud.google.com/products#product-launch-stages">Preview</a>) to search
on tables that have autonomous embedding generation enabled. You can improve
vector search performance by creating a
<a href="/bigquery/docs/vector-index">vector index</a>, which uses Approximate Nearest
Neighbor search techniques to provide faster, more approximate results.</p>

<p>Common use cases for vector search include semantic search, recommendation,
and retrieval-augmented generation (RAG). For more information, see
<a href="/bigquery/docs/vector-search-intro">Introduction to vector search</a>.</p>
</li>
</ul>

<h2 id="assistive_ai_features">Assistive AI features</h2>

<p>AI-powered assistance features in BigQuery, collectively
referred to as
<a href="/bigquery/docs/gemini-overview">Gemini in BigQuery</a>,
help you discover, prepare, query, and visualize your data.</p>

<ul>
<li>
<a href="/bigquery/docs/data-insights"><strong>Data insights.</strong></a> Generate
natural language questions about your data, along with the SQL
queries to answer those questions.</li>
<li>
<a href="/bigquery/docs/data-prep-introduction"><strong>Data preparation.</strong></a> Generate
context aware recommendations to clean, transform, and enrich your data.</li>
<li>
<a href="/bigquery/docs/write-sql-gemini#generate_a_sql_query"><strong>SQL code assist.</strong></a>
Generate, complete, and explain SQL queries.</li>
<li>
<a href="/bigquery/docs/write-sql-gemini#generate_python_code"><strong>Python code assist.</strong></a>
Generate, complete, and explain Python code, including PySpark and
BigQuery DataFrames.</li>
<li>
<a href="/bigquery/docs/data-canvas"><strong>Data canvas.</strong></a> Query your data using natural
language, visualize results with charts, and ask follow-up questions.</li>
<li>
<a href="/bigquery/docs/interactive-sql-translator"><strong>SQL translator.</strong></a> Create
Gemini-enhanced SQL translation rules to help you migrate
queries written in a different dialect to GoogleSQL.</li>
</ul>

<h2 id="agents">Agents</h2>

<p>Agents are software tools that can use AI to complete tasks on your behalf.
You can use built-in agents or create your own agents to help you process,
manage, analyze, and visualize your data:</p>

<ul>
<li><p>Use the <a href="/bigquery/docs/colab-data-science-agent">Data Science Agent</a> to
automate exploratory data analysis, data processing, ML tasks,
and visualization insights within a Colab Enterprise notebook.</p></li>
<li><p>Use the
<a href="/bigquery/docs/data-engineering-agent-pipelines">Data Engineering Agent</a>
to build, modify, and manage data pipelines to load and process data in
BigQuery. You can use natural language prompts to generate data
pipelines from various data sources or adapt existing data pipelines to suit
your data engineering needs.</p></li>
<li><p>Use the
<a href="/bigquery/docs/conversational-analytics">Conversational Analytics Agent</a>
to chat with your data using conversational language. This agent consists of
one or more data sources and a set of use case-specific instructions for
processing that data. Conversation analytics supports the use of <a href="/bigquery/docs/conversational-analytics#bigquery-ml-support">some
BigQuery ML functions</a>.</p></li>
<li><p>Use the <a href="/bigquery/docs/develop-with-gemini-cli">Gemini CLI</a> to
interact with BigQuery data in your terminal by using
natural language prompts.</p></li>
<li><p>Build using the <a href="/bigquery/docs/pre-built-tools-with-mcp-toolbox">Open source MCP toolbox</a>
or <a href="https://google.github.io/adk-docs/integrations/bigquery/">ADK tools</a> for
quick, iterative agent development.</p></li>
</ul>
