Cloud Customer Care offerings
Support offerings are available from Google Cloud that can include 24/7 coverage,
phone support, and access to technical account services. For more information, see
Getting Cloud Customer Care .
Request quota changes
To increase or decrease quotas, see
Quotas and limits .

community on Stack Overflow with the tag
google-bigquery .

File feature requests and bug reports
If you'd like to request a new feature for BigQuery, or encounter a reproducible
bug, you can report them through the
issue tracker . You can
review the open issues to see
if your bug or request has already been reported, and star the existing thread to keep updated.
If there's no existing thread, you can
request a new feature
or
create new bug .
